Simple Process

How GSTSyncPro Works

From file upload to a complete ITC summary in four easy steps. All processing happens locally in your browser — your data never leaves your device.

1

Upload 4 Files

Open GSTSyncPro in Chrome and drag & drop (or click to browse) your four data files: GSTR-2B Purchase data, Books Purchase register, GSTR-2B CN/DN data, and Books CN/DN register. Supported formats: Excel (.xlsx, .xls) and CSV.

2

Auto-Reconcile

Click "Reconcile" and GSTSyncPro automatically runs a 3-pass purchase matching algorithm (Exact → Fuzzy → Amount-based) and a 6-pass CN/DN matching algorithm (Cross-type + Same-type). Each pass catches records the previous pass missed.

3

Review & Act

Review each record in the results table. Use per-record actions — Accept matched pairs, Unmatch incorrect pairs, manually Pair records, Force Pair with overrides, or mark as Not Matched. Bulk select-all and GSTIN-filter make large datasets manageable.

4

Export Results

When done, export your reconciliation as a formatted Excel file for filing, a JSON file for ERP integration, or use the Copy ITC Summary button to paste the net claimable ITC figure directly into your GST return workflow.

Data Privacy

Your GST Data Never Leaves Your Browser

All file parsing, matching, and ITC calculation happens entirely inside Chrome using local JavaScript. Only license validation communicates with our server — no GSTIN, invoice, or financial data is ever transmitted.
